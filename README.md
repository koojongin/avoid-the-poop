## Initialize for dev update

```
$ python -m virtualenv venv  
$ (venv) pip install pygame 
$ pip freeze > requirements.txt
```

## Getting started
Windows
```
$ python -m virtualenv venv
$ pip install -r requirements.txt 
$ python index.py
```
